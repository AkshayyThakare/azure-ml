## Credit Card Fraud Detection (Azure ML)

### Overview
Built a machine learning model to detect fraudulent credit card transactions using Python and Microsoft Azure Machine Learning.

### Tech Stack
- Python
- Microsoft Azure ML
- Pandas, NumPy, Scikit-learn
- Matplotlib
- PowerPoint (visual reporting)

### Workflow
- Data ingestion using Azure storage
- Exploratory data analysis
- ML model training and evaluation
- Achieved ~98% accuracy
- Performance visualization

### Key Outcomes
- Identified fraudulent vs normal transactions
- Visualized class imbalance and model accuracy
- Deployed and executed pipeline in Azure ML
